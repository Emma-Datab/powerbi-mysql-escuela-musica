# 📊 Análisis de Datos - Escuela de Música

## Descripción
Base de datos en MySQL y dashboard en Power BI para análisis de una escuela de música.

## Tecnologías utilizadas
- **MySQL / MariaDB** — Diseño y gestión de base de datos relacional
- **SQL** — Consultas avanzadas: JOINs, subconsultas, window functions, DAX
- **Power BI** — Dashboard interactivo con KPIs y visualizaciones
- **phpMyAdmin** — Administración de la base de datos

## Estructura de la base de datos
- `alumnos` — 80 registros
- `cursos` — 15 cursos por instrumento
- `inscripciones` — 148 registros con fechas y estado de pago

## Dashboard
El dashboard incluye:
- KPIs: Total alumnos, inscripciones, cursos e ingresos totales
- Inscripciones por mes (gráfico de líneas)
- Alumnos por instrumento (gráfico de anillos)
- Ingresos por instrumento (barras)
- Inscripciones por ciudad (barras)
- Últimas inscripciones (tabla)
